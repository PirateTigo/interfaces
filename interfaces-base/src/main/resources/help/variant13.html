Вариант 13

Список дел

Имеется поле ввода элемента списка. Кнопка «Добавить». Список, отображающий введенные дела. Напротив каждого элемента списка имеется чек-бокс для выбора элемента. Кнопка «Удалить». Кнопка «Выполнить». При нажатии на кнопку «Добавить» из поля ввода элемента в список добавляется введенная строка. При нажатии на кнопку «Удалить» из списка удаляются выбранные элементы. При нажатии на кнопку «Выполнить» выбранные элементы зачеркиваются.
