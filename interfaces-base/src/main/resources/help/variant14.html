Вариант 14

Сортировщик списков

Имеется кнопка «Загрузить», при нажатии на которую пользователю предоставляется возможность выбрать файл с диска, далее содержимое файла загружается в текстовую область отображения строк файла. Имеется кнопка «Сортировать», при нажатии на которую содержимое текстовой области сортируется в порядке возрастания. При повторном нажатии список сортируется в обратном порядке. Имеется кнопка «Сохранить», при нажатии на которую пользователю предоставляется возможность сохранить файл на диск.
