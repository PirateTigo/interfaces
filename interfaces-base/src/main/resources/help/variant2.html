Вариант 2

Разложение на произведение простых чисел

Назначение программы:

Программа реализует возможность разложения значения числа на произведение простых чисел.

Описание функциональных возможностей:

Пользователю предоставляется возможность ввода числа с помощью поля ввода. Программа выполняет расчет простых чисел и их степеней, затем выводит на экран множители и их степени.
