Вариант 7

Поле Чудес

Пользователь вводит строку в поле ввода и нажимает кнопку «Игра», программа отображает черные квадраты, соответствующие символам введенной строки (кроме пробельных). Пользователь нажимает кнопки на клавиатуре. Если введенный пользователем символ присутствует во введенной ранее строке, то черные квадраты, соответствующие позиции символа заменяются этим символом.
