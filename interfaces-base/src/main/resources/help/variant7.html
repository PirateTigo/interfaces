Вариант 7

Поле Чудес

Назначение программы:

Программа представляет собой вариант известной одноименной телевизионной игры.

Описание функциональных возможностей:

Пользователь имеет возможность указать набор строк, одна из которых случайным образом будет выбрана в качестве загадываемой после начала игры. Пользователь запускает игру. Программа отображает черные квадраты, соответствующие символам загаданной строки (кроме пробельных). Пользователь нажимает кнопки на клавиатуре. Если введенный пользователем символ присутствует в загаданной строке, то черные квадраты, соответствующие позиции символа заменяются этим символом. Игра завершается, когда пользователь открыл все буквы загаданной строки.
