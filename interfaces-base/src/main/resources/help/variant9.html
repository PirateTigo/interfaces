Вариант 9

Расчет объемов геометрических тел

Пользователь с помощью радиокнопок выбирает тип тела (куб, пирамида, призма, сфера, конус, цилиндр, параллелепипед и пр.). При выборе тела на экране отображаются поля ввода параметров тела. После ввода значений пользователь нажимает на кнопку «Объем» и программа вычисляет объем выбранного тела с указанными параметрами.
