Вариант 9

Расчет объемов геометрических тел

Назначение программы:

Программа представляет собой калькулятор для расчета объемов некоторых геометрических тел.

Описание функциональных возможностей:

Пользователь с помощью группы радиокнопок выбирает вид тела (куб, пирамида, призма, сфера, конус, цилиндр, параллелепипед). После выбора тела пользователь указывает параметры данного тела. Затем, пользователь нажимает на кнопку «Объем» и программа вычисляет объем выбранного тела с указанными параметрами.
