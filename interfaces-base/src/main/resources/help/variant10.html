Вариант 10

Расчет площадей геометрических фигур

Пользователь с помощью радиокнопок выбирает тип фигуры (квадрат, треугольник, трапеция, прямоугольник, окружность и пр.). При выборе фигуры на экране отображаются поля ввода параметров фигуры. После ввода значений пользователь нажимает на кнопку «Площадь» и программа вычисляет площадь выбранной фигуры с указанными параметрами.
