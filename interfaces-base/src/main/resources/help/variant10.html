Вариант 10

Расчет площадей геометрических фигур

Назначение программы:

Программа представляет собой калькулятор для расчета площадей некоторых геометрических фигур.

Описание функциональных возможностей:

Пользователь с помощью группы радиокнопок выбирает тип фигуры (квадрат, треугольник, трапеция, прямоугольник, окружность, ромб, параллелограмм). После выбора фигуры пользователь указывает параметры данной фигуры. Затем, пользователь нажимает на кнопку «Площадь» и программа вычисляет площадь выбранной фигуры с указанными параметрами.
