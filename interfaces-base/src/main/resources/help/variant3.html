Вариант 3

Следующее простое число

Назначение программы:

Программа реализует возможность расчета простого числа, следующего за указанным.

Описание функциональных возможностей:

Пользователю предоставляется возможность ввода числа в поле ввода. После ввода на экран выводится простое число, большее введенного. При следующей попытке ввода рассчитывается следующее простое число и т.д.
