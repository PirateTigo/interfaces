Вариант 11

Вычисление параметров части электрической цепи с пассивными элементами

Пользователь может добавить на экран элементы резисторы (прямоугольники) и указать значения их сопротивлений. Программа указывает точку входа подключения и точку выхода, которые пользователь может подключить к элементам с помощью «проводов». После «подключения» пользователь указывает значения силы тока I, напряжения U и нажимает кнопку «Рассчитать», программа выводит на экран значение сопротивления данного участка цепи.
