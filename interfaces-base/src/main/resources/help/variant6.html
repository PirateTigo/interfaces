Вариант 6

Симуляция подбрасывания монеты

Назначение программы:

Программа позволяет пользователю выполнить эксперимент по последовательному подбрасыванию монеты несколько раз.

Описание функциональных возможностей:

Пользователю предоставляется возможность "подбросить" монету. После подбрасывания программа выводит на экран процент «выпадания» орла и решки. Последовательные подбрасывания приводят к аддитивному накоплению значений. В случае необходимости пользователь имеет возможность начать последовательность экспериментов с начала.
